Что это

Пульт ручного управления одним БАС. Команды идут браузер → MAVProxy → ns-3 (радиоканал) → ArduPilot SITL. Это пульт, а не витрина состояния — для мониторинга см. Admin Dashboard :8810.

Порядок взлёта

GUIDED → ARM (или FORCE) → TAKEOFF → лети WASD/Space/Ctrl или GO TO → LAND.

Клавиши

W/A/S/D — горизонт, Space — вверх, Ctrl — вниз, Esc — стоп. Удерживайте для непрерывного движения.

Панели

Command Map — позиция в локальном NED + цель GO TO. RF Link — RSSI/loss/delay, растёт при заходе за здание (NLOS). FPV — видео с борта. Events — лог команд.
