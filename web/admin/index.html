Что показывает этот экран

Это административная витрина стенда: она показывает состояние ИССГР, бортовой БД, multicast-синхронизации, картографического тайлинга и объектов БАС. Полётные команды отправляются не отсюда, а через Web GCS или QGroundControl.

Для чего

Показать, что модули грантового стенда связаны в одну систему: наземная ИССГР, бортовая БД, синхронизация, карта, БАС-объекты и диагностические события.

Stage 3 multicast sync

Синхронизация ИССГР между узлами

Publisher читает node-A, кодирует heartbeat и позиции БАС в compact UDP multicast packets, subscriber реплицирует их в node-B.

Protocol contract

Что закрывает требование синхронизации БД
Endpoint239.10.10.10:5500
TTL1, admin-local multicast scope
Packet sizesL1=40B position, L2=80B sensor
IntegrityFNV-1a identifiers + CRC-16/CCITT-FALSE
